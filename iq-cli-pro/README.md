## My:IQ - CLI Pro

My:IQ is a cognitive-intelligence platform. This repository hosts Apache Camel components for My:IQ.

This module hosts the advanced CLI tools.

### Getting Started

This repository  builds an executable JAR containing the CLI.

### Prerequisites

```
    mvn clean install
```
